# Mahdi Podcast CMS

افزونه اختصاصی وردپرس برای مدیریت پادکست با پشتیبانی کامل از فصل‌ها، پلیر حرفه‌ای و سئو.

---

## نصب

1. پوشه `mahdi-podcast-cms` را در `/wp-content/plugins/` آپلود کنید.
2. از پیشخوان وردپرس → افزونه‌ها → فعال‌سازی کنید.
3. یک صفحه جدید بسازید و شورت‌کد را در آن قرار دهید:

```
[mahdi_podcast]
```

---

## ساختار فایل‌ها

```
├── mahdi-podcast-cms.php       ← فایل اصلی افزونه

├── admin/

│   ├── settings.php            ← صفحه تنظیمات
│   ├── css/admin.css
│   └── js/admin.js
├── public/
│   ├── css/frontend.css        ← استایل‌های صفحه پادکست
│   └── js/
│       ├── player.js           ← موتور پلیر صوتی
│       └── frontend.js         ← تعاملات صفحه

    └── default-cover.svg
```

## شورت‌کد

```
[mahdi_podcast]
```

### پارامترهای اختیاری

| پارامتر | مقدار پیش‌فرض | توضیح |
|---------|--------------|-------|
| `season` | (همه) | فقط یک فصل نمایش داده شود (شناسه term) |
| `limit` | -1 (همه) | تعداد اپیزود |
| `order` | ASC | ترتیب نمایش |
| `show_search` | 1 | نمایش جستجو |
| `show_filter` | 1 | نمایش فیلتر فصل |

## امکانات

- ✅ مدیریت فصل‌ها با تصویر، رنگ و توضیحات
- ✅ فیلدهای کامل اپیزود (صوت، مدت، شماره، نوع، ...)
- ✅ پلیر sticky پایین صفحه با صف پخش
- ✅ آکاردئون فصل‌ها
- ✅ جستجو و فیلتر با AJAX
- ✅ Lazy Load تصاویر
- ✅ Schema.org برای سئو پادکست
- ✅ ریسپانسیو کامل
- ✅ کیبورد شورت‌کات (Space = پخش/توقف)

---

## تنظیمات

پیشخوان → پادکست → تنظیمات

- رنگ طلایی، کرم، تیره (قابل تغییر)
- اطلاعات پادکست برای سئو
- نمایش/مخفی کردن بخش‌ها
